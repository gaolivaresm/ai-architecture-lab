# Aprendizaje Máquina para Arquitectos — TAETT

Programa de Mentoría Técnica avanzada sobre la integración de Inteligencia Artificial, Ciencia de Datos y Automatización en la práctica profesional de la Arquitectura y la Ingeniería.

## 🎯 Objetivo del Programa
Transferir metodologías probadas para la automatización de flujos de trabajo críticos, desde el análisis territorial hasta la gestión documental inteligente (RAG) y la automatización BIM.

## 🛠️ Stack Técnico
- **Lenguajes:** Python, SQL.
- **IA/ML:** RAG (Retrieval-Augmented Generation), SciML, Diffusion Models.
- **BIM:** Revit, Dynamo, Grasshopper.
- **Entorno:** Google Colab, Linux HPC, Docker.

## 🚀 Estructura del Taller
1. **S1:** Arquitectura de IA + Análisis Exploratorio (EDA).
2. **S2:** Integración de Datos y APIs Urbanas.
3. **S3:** Inteligencia Documental con RAG (Memoria de Oficina).
4. **S4:** IA Generativa y Control Geométrico.
5. **S5:** BIM-Automation + Cloud Ecosystem.
